# Utiliser Claude Code avec Ollama (100% local, 0 API key)

## Pourquoi ça marche

Ollama v0.14+ implémente l'API Anthropic Messages (`/api/anthropic/v1/messages`).
Claude Code utilise `ANTHROPIC_BASE_URL` pour rediriger ses appels vers n'importe quel serveur.

## Configuration en 3 étapes

### 1. Installer Ollama + le bon modèle

```bash
# Installer Ollama
brew install ollama   # macOS
# ou : curl -fsSL https://ollama.ai/install.sh | sh

# Télécharger qwen2.5-coder (meilleur modèle code open source 2026)
ollama pull qwen2.5-coder:32b   # Mac M2 16GB : ok
# ou version légère :
ollama pull qwen2.5-coder:7b    # 4GB RAM minimum
```

### 2. Créer un modèle avec contexte élargi (OBLIGATOIRE)

Claude Code envoie de longs contextes. Sans ça, le modèle coupe à 4096 tokens.

```bash
cat > /tmp/Modelfile-chimera << 'EOF'
FROM qwen2.5-coder:32b
PARAMETER num_ctx 65536
PARAMETER temperature 0.2
SYSTEM "You are an expert software engineer helping with the Chimera OS project. Be precise and autonomous."
EOF

ollama create chimera-coder -f /tmp/Modelfile-chimera
```

### 3. Lancer Claude Code en mode Ollama

```bash
# Dans le terminal, avant de lancer claude :
export ANTHROPIC_API_KEY=ollama
export ANTHROPIC_BASE_URL=http://localhost:11434

# Puis :
claude --model chimera-coder

# Ou en one-liner :
ANTHROPIC_API_KEY=ollama ANTHROPIC_BASE_URL=http://localhost:11434 claude
```

## Ajouter à ton ~/.zshrc pour basculer facilement

```bash
# Ajoute à ~/.zshrc :

# Mode Claude (Anthropic officiel)
alias claude-cloud='unset ANTHROPIC_BASE_URL && claude'

# Mode Ollama (100% local)
alias claude-local='ANTHROPIC_API_KEY=ollama ANTHROPIC_BASE_URL=http://localhost:11434 claude'
alias cl='claude-local'  # raccourci

# Mode Chimera (Ollama + modèle spécialisé)
alias chimera-code='ANTHROPIC_API_KEY=ollama ANTHROPIC_BASE_URL=http://localhost:11434 claude --model chimera-coder'
```

## Modèles recommandés pour Chimera

| Modèle Ollama | Usage | RAM | Qualité |
|---|---|---|---|
| `qwen2.5-coder:32b` | Code principal | 20GB | ⭐⭐⭐⭐⭐ |
| `qwen2.5-coder:7b` | Code léger | 6GB | ⭐⭐⭐⭐ |

| `llava:13b` | Vision rapide | 8GB | ⭐⭐⭐ |
| `deepseek-r1:14b` | Raisonnement | 9GB | ⭐⭐⭐⭐ |

## Bug connu

L'endpoint `/v1/messages/count_tokens` peut retourner une erreur 500 sur certaines
versions d'Ollama. Si Claude Code crashe, mettre à jour Ollama :

```bash
brew upgrade ollama   # macOS
ollama --version     # doit être >= 0.14.0
```

## Alternative : OpenCode (open source, même expérience)

Si Claude Code + Ollama pose des problèmes, utilise OpenCode :

```bash
npm i -g opencode-ai@latest

# Config ~/.config/opencode/opencode.json :
{
  "providers": {
    "ollama": {
      "baseURL": "http://localhost:11434/v1",
      "type": "openai-compatible"
    }
  },
  "model": "ollama/chimera-coder"
}

opencode  # dans le dossier chimera/
```
